## 1. 风电场景概况

风电故障与现有产业（通用/锂电/光伏/储能）有显著差异：
- **旋转机械为主**: 齿轮箱、轴承、发电机、叶片是主要故障源
- **连续振动信号**: 故障通过 FFT 频谱/包络谱识别（非离散阈值）
- **气象因素**: 风速、湍流强度、结冰条件直接影响故障特征
- **远程运维**: 海上风电故障修复需远程诊断 + 陆运调配

## 2. 建议故障模板 (F-051~F-060)

| ID | 名称 | Severity | 建议策略 | 说明 |
|---|---|---|---|---|
| F-051 | gearbox_tooth_crack | P1 | S3 | 齿轮箱齿面裂纹 → 紧急停机 + 远程专家介入 |
| F-052 | main_bearing_temp_anomaly | P2 | S1 | 主轴轴承温升超阈值 |
| F-053 | blade_icing_detected | P1 | S2 | 叶片结冰 → 降功率运行 + 除冰系统激活 |
| F-054 | generator_winding_temp | P2 | S1 | 发电机定子绕组温升 |
| F-055 | tower_vibration_excess | P1 | S3 | 塔筒振动超限 → 紧急停机 |
| F-056 | pitch_system_failure | P2 | S1 | 变桨系统执行器卡滞 |
| F-057 | converter_igbt_overheat | P2 | S2 | IGBT 模块温度超限（需备件） |
| F-058 | lightning_strike_event | P1 | S3 | 雷击事件 → 全面巡检 + 保险流程 |
| F-059 | scada_communication_loss | P2 | S1 | SCADA 通信链路丢失 |
| F-060 | offshore_cable_corrosion | P3 | S2 | 海缆腐蚀预警（需定期检查） |

## 3. v4.0 产业兼容性评估

| 维度 | 风电适配性 | 说明 |
|---|---|---|
| category | ✅ 现有 4 类够用 | `equipment_fault`(齿轮箱/轴承), `process_anomaly`(振动/结冰) |
| severity | ✅ P1/P2/P3 适用 | 安全停机 → P1, 降功率 → P2, 预警 → P3 |
| signals | ⚠️ 需新增频域信号 | 当前信号格式是离散键值对，风电需要 FFT 频谱特征 |
| repair_strategy | ✅ S1/S2/S3 适用 | 参数调整 S1 / 备件确认 S2 / 停机+专家 S3 |
| industry | ✅ energy_storage 不可用 | 需新增 `wind_power` industry 枚举 |

## 4. 风险评估

| 风险 | 概率 | 影响 | 缓解 |
|---|---|---|---|
| 连续信号与离散模板不匹配 | 高 (70%) | 中 | 信号格式改为 `key: value` 字符串，频域特征存为标记 |
| 海上风电远程修复延迟 | 中 (40%) | 高 | S3 escalation 加入远程诊断路径 |
| 无新产业模板经验 | 中 (50%) | 低 | 已有锂电/光伏/储能经验，流程可复用 |
